💡 Concept Overview

บอร์ดเกมแนวแข่งขันวางแผนครองพื้นที่ในอวกาศ ผู้เล่นรับบทเป็นนักสำรวจช่วงชิงอำนาจยึดครองดวงดาว บริหารพลังงานเหรียญ และหลบเลี่ยงภัยธรรมชาติอย่างหลุมดำเพื่อสะสมดาวเคราะห์ให้ได้คะแนนสูงสุด ก่อนที่จักรวาลจะถึงจุดดับสลายจากการถูกเปิดการ์ดบิ๊กแบง

⚙️ Core System Design Specifications

1. Modular Board Architecture (กระดานโมดูลาร์เปลี่ยนฉาก)

ออกแบบระบบการวางแผนผังกระดานแบบสลับสับเปลี่ยนชิ้นส่วนได้ (Modular Board Grid) แผนที่กระดานจะถูกจัดวางใหม่โดยระบบสุ่มทุก ๆ เริ่มต้นเกมใหม่ เพื่อสร้าง Replayability สูง และท้าทายจังหวะการมองหาชัยภูมิของผู้เล่น

2. Core Game Loop Logic (ตรรกะวงจรการเล่นหลัก)

วงจรเทิร์นในเกมนอกเหนือจากการทอยเต๋า จะประมวลผลผ่าน 3 ลำดับขั้น:

Income & Action Phase: คำนวณรายได้จากอาณาเขตดาวที่ยึดได้ และอนุญาตการใช้ไอเทม

Roll & Move Phase: ควบคุมพิกัดขยับยูนิตตามลูกเต๋าและการจัดแนวเดินเลี่ยงสิ่งกีดขวาง

Encounter Phase: การปะทะพื้นที่ แย่งชิงดาว แย่งชิงทรัพยากร หรือเปิดกล่องสัมภาระพิเศษ

3. Balanced Game Economy (ระบบควบคุมความสมดุลเศษฐศาสตร์)

Resource Balancing & Risk: การบริหารจำนวนเหรียญ (Coins) และการ์ดไอเทมเพื่อประหยัดต้นทุนหรือสร้างความได้เปรียบ

Math Balancing from 100+ Simulations: ทำการคำนวณและปรับลด-เพิ่มสมรรถภาพการสุ่มความน่าจะเป็นของการ์ดสถานการณ์ (Event Cards) เพื่อให้โอกาสการเกิดผลบวกและผลลบไม่เอนเอียง และผ่านการคำนวณอัตราความคุ้มทุนของการซื้อสิทธิ์ยึดดาวเคราะห์เทียบกับการสะสมแต้ม
